# LLM Translator (LLM 翻译助手)

利用大语言模型 (LLM) 的强大功能来翻译网页并高亮显示词汇。这款 Chrome 扩展程序允许您使用自己喜欢的 LLM 提供商（OpenAI 或任何兼容 OpenAI 的 API）直接在浏览器中翻译文本、分析词汇等。

## 功能特性

- **右键菜单翻译**：选中任意文本，右键点击并选择“Translate selection”（翻译选中内容），即可在可拖动的弹窗中获得即时翻译。
- **悬浮图标**：选中文本后点击出现的悬浮图标，可快速进行翻译。
- **行内翻译**：可选择直接在原文下方显示翻译结果，提供流畅的阅读体验。
- **多模型支持**：配置多个 LLM 提供商和模型。可以对比不同模型的结果，或针对不同任务使用不同的模型。
- **词汇高亮**：（实验性功能）分析并高亮显示页面上的高级词汇（CEFR B2/C1/C2 等级），并提供定义。
- **自定义提示词**：微调用于翻译和词汇提取的系统提示词，以满足您的特定需求。
- **现代 UI**：简洁、“高级”的视觉设计，支持深色模式元素，交互流畅。

## 安装指南

1.  克隆或下载本仓库。
2.  打开 Chrome 浏览器并导航至 `chrome://extensions/`。
3.  开启右上角的 **Developer mode**（开发者模式）。
4.  点击 **Load unpacked**（加载已解压的扩展程序）。
5.  选择包含本扩展程序的目录。

## 配置说明

1.  点击 Chrome 工具栏中的扩展程序图标，选择 **Options**（选项）（或者右键点击图标并选择“选项”）。
2.  **General Settings**（通用设置）：
    *   **Target Language**（目标语言）：选择您希望翻译成的目标语言。
    *   **Custom Prompts**（自定义提示词）：（可选）覆盖默认的翻译和词汇分析提示词。
    *   **Inline Mode**（行内模式）：启用/禁用行内翻译功能。
3.  **Model Profiles**（模型配置）：
    *   添加一个或多个 LLM 提供商。
    *   **Provider**（提供商）：选择 "OpenAI" 或 "Custom"（自定义，适用于 Ollama、LM Studio 等本地 LLM 或其他兼容 API）。
    *   **API Endpoint**（API 端点）：聊天补全接口的完整 URL（例如 `https://api.openai.com/v1/chat/completions`）。
    *   **API Key**（API 密钥）：您的 API 密钥。
    *   **Model Name**（模型名称）：要使用的具体模型名称（例如 `gpt-4o`, `gpt-3.5-turbo`, `llama3`）。
4.  点击 **Save Settings**（保存设置）。

## 使用方法

### 翻译文本
1.  在网页上高亮选中任意文本。
2.  **右键点击**并选择 **Translate selection**。
3.  或者，点击选中区域附近出现的**悬浮图标**。
4.  将会弹出一个包含翻译结果的弹窗。您可以拖动弹窗将其移开。

### 行内翻译
如果在设置中启用了该功能：
1.  高亮选中文本。
2.  点击 **Inline Translate**（行内翻译）图标（如果显示）或使用右键菜单。
3.  翻译结果将直接插入到文本流中。

## 隐私说明

本扩展程序直接与您配置的 LLM API 端点进行通信。您的 API 密钥仅存储在您浏览器的本地同步存储中。除了您明确配置的 LLM 提供商之外，不会向任何第三方服务器发送数据。
